slisguid

PURPOSE

SLISGUID Judges whether the input can represent a GUID

SYNOPSIS

function b = slisguid(v)

DESCRIPTION 

SLISGUID Judges whether the input can represent a GUID

 $ Syntax $
   - b = slisguid(v)

 $ Description $
   - b = slisguid(v) judges whether the input v conforms to the 
     matlab representation of GUID (1 x 16 uint8 array).
