slguidstr

PURPOSE

SLGUIDSTR Converts a guid to a string

SYNOPSIS

function str = slguidstr(gid)

DESCRIPTION 

SLGUIDSTR Converts a guid to a string 

 $ Syntax $
   - str = slguidstr(gid)

 $ Description $
   - str = slguidstr(gid) converts a GUID represented by a 1 x 16 uint8
     array to a string in Win32 Registry Format.
